# Antum Passives mod for Minetest

An attempt at compiling passive mobiles from various mods for [Minetest][].

Please note, this project is a work-in-progress & is not yet funcitonal.

---
### Dependencies

- ***mob_sheep***
  - Required:
    - [creatures][]
    - [default][]
  - Optional:
    - [farming][]
    - [wool][]
